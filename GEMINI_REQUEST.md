## ✅ 詳しい解説の書き方（重要）

### 必須要素

各政策の「詳しい解説」には、以下を**必ず含めてください**：

1. **専門用語の説明**
   - 例：「診療報酬」→「病院・クリニックが患者さんを診察したときに受け取る報酬」
   - 例：「配置基準」→「病院に何人のスタッフが必要かという基準」

2. **具体的な金額・数字**
   - 例：「月額10万円の給付」「最大500万円の補助金」「3割削減」

3. **いつから実施するか**
   - 例：「2026年4月から」「段階的に導入」「即時実施」

4. **誰にどんな影響があるか**
   - 例：「医学生の借金の心配が減ります」
   - 例：「患者さんの待ち時間も短縮されます」
   - 例：「月給で3〜5万円程度の改善を目指します」

5. **絵文字で視覚的に分かりやすく**
   - 医学生：🎓
   - 開業医：💊
   - 勤務医：⏰
   - 言語聴覚士：🗣️
   - 理学療法士：🏃
   - ケアマネージャー：📋
   - 消費税：🛒
   - 投資・経済：💰
   - 子育て：👶
   - エネルギー：⚡
   - ペット：🐾

## 📝 解説の良い例・悪い例

### ❌ 悪い例（専門用語だらけ、影響不明）

```
"診療報酬の物価スライド導入検討。医療DX導入費用への補助金拡充と税制優遇。"
```

### ✅ 良い例（専門用語を説明、具体的、影響明確）

```
💊 **診療報酬**は病院・クリニックが患者さんを診察した時に受け取る報酬のこと。物価が上がれば自動的に診療報酬も上がる「物価スライド制」を検討します。また、電子カルテなどデジタル化の設備投資に最大500万円の補助金と税金の優遇措置を実施。患者さんの待ち時間も短縮されます。
```

## ⚠️ 注意事項

1. **中立性を保つ**
   - 特定政党を有利/不利にするような書き方をしない
   - 事実ベースで記載

2. **最新情報を優先**
   - 2026年選挙に向けた最新の公約を使用
   - 過去の政策ではなく、現在の方針を反映

3. **わかりやすさ最優先**
   - 小学6年生でも理解できる表現
   - 専門用語には必ず補足説明

4. **具体性を重視**
   - 「検討します」だけでなく、具体的な金額・時期・方法
   - 抽象的な表現は避ける
